# 🤖 AI Persona Bot

This is a WhatsApp AI Chatbot application that creates an interactive conversational AI assistant. 

The system allows users (i.e., your contacts) to chat with an AI persona of you via WhatsApp and receive responses styled as your own.

## ❓ FAQs

### How to set up a Twilio Sandbox for testing?

Refer to the official Twilio guide:  
https://www.twilio.com/docs/whatsapp/quickstart

1. Set up your Twilio account and WhatsApp Sandbox  
2. Obtain your Twilio credentials (Account SID & Auth Token)  
3. Add them to your environment variables  
4. Run the application
